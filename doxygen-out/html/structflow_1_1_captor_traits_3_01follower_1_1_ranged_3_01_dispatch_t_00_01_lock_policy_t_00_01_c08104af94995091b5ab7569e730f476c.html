Detailed Description

template<typename DispatchT, typename LockPolicyT, typename ContainerT, typename QueueMonitorT>
struct flow::CaptorTraits< follower::Ranged< DispatchT, LockPolicyT, ContainerT, QueueMonitorT > >

Traits struct for captor types.

Requires:

  • DispatchType : data dispatch object type
  • DispatchContainerType : container for DispatchType DispatchQueueMonitorType : optional monitoring/capture preconditioning object type
  • value_type : data value type
  • stamp_type : sequence stamp type
  • size_type : integer sizing type
