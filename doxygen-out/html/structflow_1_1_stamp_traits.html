Detailed Description

template<typename StampT>
struct flow::StampTraits< StampT >

Helper struct used to specify stamp attributes.

Must be specialized for non-integral sequence stamp types with the same fields:

  • min : the minimum value of StampT
  • max : the maximum value of StampT
  • offset_type : associated offset/duration type
