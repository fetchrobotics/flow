Detailed Description

template<typename DispatchT, typename LockPolicyT = NoLock, typename ContainerT = DefaultContainer<DispatchT>, typename QueueMonitorT = DefaultDispatchQueueMonitor>
class flow::driver::Chunk< DispatchT, LockPolicyT, ContainerT, QueueMonitorT >

Captures the next oldest data element.

Establishes a sequencing range with where range.lower_stamp is the stamp of the oldest captured element, and range.upper_stamp is the stamp of the newest. Removes all captured elements from buffer.

Template Parameters
DispatchTdata dispatch type
LockPolicyTa BasicLockable (https://en.cppreference.com/w/cpp/named_req/BasicLockable) object or NoLock or PollingLock
ContainerTunderlying DispatchT container type
