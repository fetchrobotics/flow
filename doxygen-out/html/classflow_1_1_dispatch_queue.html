Detailed Description

template<typename DispatchT, typename ContainerT = std::deque<DispatchT>>
class flow::DispatchQueue< DispatchT, ContainerT >

Dispatch queuing data structure.

FILO-type queue which orders data by sequence stamp, from oldest to newest. Provides useful methods for extracting data within stamped/counted ranges
This template provides an interface wrapper around a specifiable container implementation.
