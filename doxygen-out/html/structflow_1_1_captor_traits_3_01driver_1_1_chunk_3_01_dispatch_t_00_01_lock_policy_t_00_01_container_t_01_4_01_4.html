Detailed Description

template<typename DispatchT, typename LockPolicyT, typename ContainerT>
struct flow::CaptorTraits< driver::Chunk< DispatchT, LockPolicyT, ContainerT > >

Traits struct for captor types.

Requires:

  • DispatchType : data dispatch object type
  • DispatchContainerType : container for DispatchType
  • value_type : data value type
  • stamp_type : sequence stamp type
  • size_type : integer sizing type
